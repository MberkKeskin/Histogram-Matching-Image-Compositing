# Histogram Matching & Image Compositing

This project performs histogram matching between two images and allows ROI (Region of Interest) based object compositing using classical computer vision techniques. It aligns color distributions between images and visualizes RGB histograms before and after matching.

---

## Features
- RGB histogram visualization  
- ROI selection using OpenCV  
- Histogram matching (color transfer)  
- Object compositing onto a reference image  
- Automatic result saving  
- Command-line interface (CLI) support
